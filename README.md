## Название проекта

**«Разработка чат-бота для поиска вакансий»**.

Программный комплекс предназначен для диалогового семантического поиска вакансий с сайта `rabota.by`: пользователь формулирует запрос на естественном языке, система извлекает структурированные фильтры с помощью большой языковой модели, выполняет векторный поиск по базе вакансий (эмбеддинги, PostgreSQL + расширение `pgvector`) и отдаёт релевантные объявления через REST API (FastAPI) и веб-интерфейс (Streamlit).

---

## Требования

- **Python** 3.10+ (рекомендуется актуальная стабильная ветка 3.x; при необходимости допускается 3.9+ с проверкой совместимости зависимостей)
- **pip** — установка зависимостей из `requirements.txt` в каталогах backend и frontend проекта `ai-job-finder-python`
- **PostgreSQL** с расширением **pgvector** — для хранения вакансий и векторного индекса (инфраструктурное требование)

Дополнительно для работы с LLM через API: переменные окружения с ключами (например, Groq и др. – по инструкции к репозиторию).
